# Rashberry

A video-first mobile app built with SolidJS + Capacitor.

## Tech Stack

- **SolidJS** - Fine-grained reactivity (no Virtual DOM overhead)
- **Vite** - Lightning-fast development server
- **Capacitor** - Native iOS/Android wrapper
- **TypeScript** - Type safety

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Mobile Development

```bash
# Sync web build to native projects
npm run cap:sync

# Open Android Studio
npm run cap:android

# Open Xcode (macOS only)
npm run cap:ios
```
